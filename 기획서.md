[기획서] 랭킹공장 - 컨셉 및 콘텐츠 전략

1. 프로젝트 아이덴티티 (Project Identity)

1.1 개요

프로젝트 명: 랭킹공장 (Rank-Factory)

슬로건: "세상 모든 것에 서열을 정리해드립니다. 반박 시 니 말이 틀림."

핵심 가치:

Raw (날것): 보정 없는 순도 100%의 리얼 랭킹.

Pop (재미): 눈이 즐겁고 타격감 있는 장난감 같은 UI.

War (논쟁): 단순 투표가 아닌, 치열한 '전쟁(Battle)' 유도.

1.2 타겟 유저 & 페르소나

프로 불편러 (The Debater): "민초가 1등? 나라 망했네"라며 댓글로 싸워야 직성이 풀리는 유저.

결정 장애 (The Hesitant): 점심 메뉴, 넷플릭스 볼거리를 남들이 정해주길 바라는 유저.

도파민 중독자 (The Scroller): 뇌를 빼고 무의미한 클릭질로 시간을 때우고 싶은 숏폼 세대.

2. 킬러 콘텐츠 라인업 (Killer Content Strategy)

단순 나열이 아닌, 감정(분노, 공감, 우월감)을 자극하는 3가지 포맷으로 운영합니다.

2.1 [Battle] 무한 밸런스 월드컵 (메인 콘텐츠)

"둘 중 하나만 골라. 선택장애 치료제."

형식: 1:1 매치 (이상형 월드컵) + ELO 랭킹 시스템.

주요 주제:

음식: [라면 계급도] 신라면(근본) vs 불닭볶음면(도파민)

직장: [최악의 상사] 꼰대 vs 히스테리

Tech: [개발자 발작] 변수명 a, b 쓰기 vs 주석 없이 1000줄 코딩

2.2 [Fact] 팩트 폭력 랭킹 (유입용)

"데이터는 거짓말 안 함. 팩트로 뼈 때려드립니다."

형식: 외부 API 및 크롤링 데이터 시각화 (자동 업데이트).

주요 주제:

Money: 실시간 비트코인 수익률 상위 코인.

Movie: 관객 평점 vs 평론가 평점 괴리율 순위 (호불호 갈리는 영화).

2.3 [Hell] 실시간 티어표 (확장용)

"S급부터 폐급까지, 니 손으로 끌어다 놓으세요."

형식: 드래그 앤 드롭으로 S/A/B/C/F 등급 배치 → 평균값 반영.

주요 주제: 프랜차이즈 치킨 티어표, 4세대 아이돌 비주얼 티어표.

3. 디자인 및 UX 철학 (Design & UX Philosophy)

3.1 디자인 컨셉: Neo-Brutalism Pop

"귀엽지만 타격감 있게."

Visual Identity:

굵은 검은색 테두리 (3px solid black).

눈을 사로잡는 원색(Yellow, Red, Purple)과 크림색 배경의 조화.

입체적인 하드 그림자(Hard Shadow)로 장난감 같은 느낌 강조.

Typography: Jua (주아체) 또는 Black Han Sans로 강렬한 임팩트 전달.

3.2 도파민 UX 전략 (Micro-Interactions)

Haptic UI (타격감): 버튼 클릭 시 꾹 눌리는 애니메이션과 모바일 진동 피드백으로 '누르는 맛' 제공.

Confetti (보상): 투표 완료 시 폭죽 효과, 연속 정답(대세 선택) 시 "Combo x5" 이펙트로 중독성 강화.

Seamless Flow (몰입): 화면 전환 없이 우측에서 슬라이드되는 댓글창(Drawer)으로 게임 흐름 유지.

4. 성장 로드맵 (Growth Strategy)

Phase 1: 기반 구축 (Week 1-2)

핵심 기능(ELO 배틀) 구현 및 초기 데이터 500개 확보 (God Mode 활용).

'라면', '치킨' 등 대중적인 주제 선점.

Phase 2: 바이럴 및 커뮤니티 (Week 3-4)

댓글 시스템(익명 닉네임) 도입으로 논쟁 유도.

"내 취향 분석 결과" 공유하기 기능(OG Image)으로 외부 유입 극대화.

Phase 3: 수익화 및 확장 (Future)

광고 매치: "최고의 편의점 맥주" 랭킹에 특정 브랜드 스폰서십 아이템 추가.

프리미엄: 닉네임 옆 '고인물' 배지 및 투표 이펙트 커스텀 판매.
